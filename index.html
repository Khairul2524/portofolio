Tentang Saya

Fullstack Developer

Saya Khairul Kahpi atau bisa dipanggil Kahpi. Saya merupakan lulusan S1 Sistem Informasi di STMIK Lombok.

Saya berkarier sebagai seorang Fullstack Developer di Dinas Komunikasi dan Informatika Kabupaten Lombok Tengah dengan fokus pada pengembangan dashboard web yang rapi, informatif, dan mudah digunakan. Ruang kerja utama saya ada pada pembangunan sistem internal, integrasi data, serta antarmuka berbasis web yang stabil dan terukur.

Spesialisasi & Tech Stack Inti

Sebagai Fullstack Developer, spesialisasi saya berfokus pada pengembangan sistem menggunakan PHP dengan pendekatan yang efisien dan siap diimplementasikan.

Backend: Laravel, CodeIgniter, RESTful API.

Data: MySQL, MongoDB, pengolahan dan integrasi data.

Frontend: HTML, CSS, JavaScript untuk dashboard dan antarmuka sistem.

Pendekatan: Clean code, visualisasi data, dan fitur berbasis machine learning.

SPOT

Lihat Disini

Aplikasi berbasis digital yang digunakan untuk mengelola dan memantau data logistik obat secara terpadu. Sistem ini membantu proses pencatatan stok obat, penerimaan, pengeluaran, distribusi, pemakaian, serta pelaporan ketersediaan obat secara lebih cepat, akurat, dan terkontrol.

Sistem ini saya bangun menggunakan framework Laravel, kemudian dikembangkan dengan model Machine Learning Prophet untuk mendukung fitur rekomendasi pengadaan obat berdasarkan data yang tersedia.

Lihat Disini
